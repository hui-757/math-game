### MapView.vue — 地图关卡

| 属性 | 值 |
|------|-----|
| 路由 | `/map/:grade/:semester` |
| 状态 | ✅ 完成 |
| 已知问题 | 无 |

**功能：**
- 从数据库动态拉取题目，按 `unit-level` 分组生成关卡
- SVG 画路线 + 关卡点组件
- 根据 `progress` 表判断关卡状态（锁定/解锁/通关）

**核心逻辑：**
```js
const all = await fetchQuestions(grade, semester)
const levelMap = new Map()
all.forEach(q => {
  const key = `${q.unit}-${q.level}`
  if (!levelMap.has(key)) {
    levelMap.set(key, { id: `${grade}-${semester}-${q.unit}-${q.level}`, ... })
  }
})
```

| 属性 | 值 |
|------|-----|
| 路由 | `/play/:levelId` |
| 状态 | ✅ 代码完成，**待手动验证** |
| 已知问题 | 无（已修复所有已知bug） |

**功能：**
- 显示当前题目（支持 4 种题型）
- 用户输入答案（原生 `<input>`）
- 计时器
- 提交后判题、保存进度、跳转结果页

**答案存储（关键！）：**
```js
// 不要改这个结构，已验证正确
const allAnswers = ref([])  // [{ b1: '17' }, { b1: '11' }, ...]

function updateAnswer(blankId, value) {
  allAnswers.value[currentIndex.value][blankId] = value
}
```

**判题逻辑：**
```js
function checkQuestion(q, userAnswers) {
  const blanks = extractBlanks(q.content)
  return blanks.every(b => {
    const user = (userAnswers[b.id] || '').trim()
    return user === b.answer.trim()
  })
}
```

### MathBlank.vue — 数学填空输入框

| 属性 | 值 |
|------|-----|
| 状态 | ⚠️ **已废弃** |
| 说明 | PlayView.vue 改用原生 `<input>` |

**废弃原因：** `v-model` 绑定 `reactive` 对象属性时 Vue 3 不追踪。

**如需复用，改为：**
```vue
<input
  :value="modelValue"
  @input="$emit('update:modelValue', $event.target.value)"
/>
```

## Store 状态（stores/）

### auth.js — 登录状态

```js
const authStore = useAuthStore()
authStore.student        // { id, class_id, nickname }
authStore.classInfo      // { id, code, name }
authStore.setStudent(data)
authStore.setClassInfo(data)
authStore.loadFromStorage()  // 从 localStorage 恢复
```

### game.js — 游戏状态

```js
const gameStore = useGameStore()
gameStore.currentQuestions   // 当前关卡的所有题
gameStore.currentIndex       // 当前第几题（0-based）
gameStore.userAnswers        // 用户答案（已废弃，用 allAnswers 代替）
gameStore.elapsedTime        // 答题用时（秒）
gameStore.lastResult         // 最近一次闯关结果

gameStore.initGame(questions, levelId)
gameStore.setAnswer(blankId, value)  // 已废弃
gameStore.nextQuestion()
gameStore.finishGame()
gameStore.setResult({ correct, total, time, mistakes })
gameStore.clearGame()
```

### progress.js — 闯关进度

```js
const progressStore = useProgressStore()
progressStore.records        // 所有闯关记录

await progressStore.load(studentId)
progressStore.isLevelCompleted(levelId)
progressStore.isLevelUnlocked(grade, semester, unit, level)
progressStore.getLevelStars(levelId)  // 0~3
```
