```sql
-- 在Supabase SQL Editor执行
CREATE TABLE classes (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  code text UNIQUE NOT NULL,
  name text NOT NULL,
  created_at timestamptz DEFAULT now()
);

CREATE TABLE students (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  class_id uuid REFERENCES classes(id),
  nickname text NOT NULL,
  created_at timestamptz DEFAULT now()
);

CREATE TABLE questions (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  grade int NOT NULL,
  semester int NOT NULL,
  unit int NOT NULL,
  unit_name text NOT NULL,
  level int NOT NULL,
  type text NOT NULL CHECK (type IN ('oral','vertical','step','equation')),
  content jsonb NOT NULL,
  answer text NOT NULL,
  difficulty int DEFAULT 1
);

CREATE TABLE progress (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  student_id uuid REFERENCES students(id),
  question_id uuid REFERENCES questions(id),
  level_id text NOT NULL,
  user_answers jsonb,
  correct_count int,
  total_count int,
  time_seconds int,
  mistakes jsonb,
  created_at timestamptz DEFAULT now()
);
```

## 六、风险登记册

| 风险 | 概率 | 影响 | 应对方案 |
|------|------|------|---------|
| AI生成的题答案错误 | 中 | **高** | Python强制验算，错误率>1%时暂停生成，修复Prompt |
| 竖式填空DOM对齐崩 | 中 | 中 | 用等宽字体+Grid，不用Canvas，降级为普通填空 |
| Supabase免费额度用完 | 低 | 高 | 提前注册，监控用量，超了换Firebase或自建SQLite |
| 教师端聚合查询慢 | 中 | 中 | 加Index，或前端做内存聚合（数据量小，可行） |
| 答辩时网络故障 | 中 | **高** | 本地开热点的手机备用，预录演示视频保底 |
| 团队成员掉线（生病/失联） | 低 | **高** | 每天git push，关键文档写README，1人可接 |
| 题库12册做不完 | 中 | 中 | 优先保证3/4/5年级（高年级展示效果好），低年补录 |
| Vue构建体积过大 | 低 | 中 | 懒加载路由，题库不走打包（走Supabase API） |
