## 七、常见问题

### Q1: 页面白屏 / Vite 编译错误
**原因：** PlayView.vue 可能有语法错误（之前出现过重复代码）  
**解决：**
```bash
# 看终端报错
npm run dev
# 找到错误行号，修复后保存，Vite 自动热更新
```

### Q2: 地图页显示"加载失败"
**原因：** Supabase URL 配置错误  
**检查：** `.env` 里的 URL 应该是 `https://jajxhtlwxphlpevmxewk.supabase.co`（**没有 `db.` 前缀**）

### Q3: 结果页显示"暂无结果"
**原因：** `gameStore.lastResult` 为空  
**检查：** PlayView.vue 的 `submitAll` 是否调用了 `gameStore.setResult()`

### Q4: 判题总是只对一个
**原因：** 之前 `reactive({})` 导致答案不更新，已修复  
**当前状态：** 改用 `ref` 数组，每题独立存储。如果还有问题，检查 `allAnswers.value[idx]` 是否有值。
