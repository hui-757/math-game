## 一、表结构

### classes — 班级

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| code | text | 班级码，如 "demo-3-2" |
| name | text | 班级名称 |
| teacher_pin | text | 教师登录 PIN |
| created_at | timestamptz | 创建时间 |

### students — 学生

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| class_id | uuid | 外键 → classes |
| nickname | text | 昵称 |
| created_at | timestamptz | 创建时间 |

### questions — 题库

| 字段 | 类型 | 说明 |
|------|------|------|
| id | text | 主键，格式 q-grade-semester-unit-level-seq |
| grade | int | 年级 1~6 |
| semester | int | 学期 1=上册, 2=下册 |
| unit | int | 单元序号 |
| unit_name | text | 单元名称 |
| level | int | 关卡序号 |
| type | text | oral / vertical / step / equation |
| content | jsonb | 题目结构 |
| answer | text | 正确答案 |
| difficulty | int | 1=简单, 2=中等, 3=困难 |
| created_at | timestamptz | 创建时间 |

### progress — 闯关记录

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| student_id | uuid | 外键 → students |
| level_id | text | 关卡ID，如 "3-2-1-1" |
| correct_count | int | 答对题数 |
| total_count | int | 总题数 |
| time_seconds | int | 用时（秒）|
| mistakes | jsonb | 错题详情 |
| created_at | timestamptz | 创建时间 |

## 二、常用查询

### 2.1 查某年级某册的所有题目

```sql
SELECT * FROM questions
WHERE grade = 3 AND semester = 2
ORDER BY unit, level;
```

### 2.2 查某学生的闯关记录

```sql
SELECT * FROM progress
WHERE student_id = 'uuid-xxx'
ORDER BY created_at DESC;
```

### 2.3 查某班级的所有学生

```sql
SELECT * FROM students
WHERE class_id = 'uuid-class'
ORDER BY nickname;
```

### 2.4 统计某班级平均正确率

```sql
SELECT
  AVG(correct_count::float / total_count) as avg_accuracy,
  COUNT(*) as total_attempts
FROM progress
WHERE student_id IN (
  SELECT id FROM students WHERE class_id = 'uuid-class'
);
```

### 2.5 统计某题错误率（易错题）

```sql
SELECT
  question_id,
  COUNT(*) as total_wrong
FROM progress,
  jsonb_array_elements(mistakes) as mistake
WHERE mistake->>'questionId' = 'q-3-2-1-1-001'
GROUP BY question_id;
```

### 2.6 统计各单元正确率（薄弱知识点）

```sql
SELECT
  q.unit,
  q.unit_name,
  AVG(p.correct_count::float / p.total_count) as avg_accuracy
FROM progress p
JOIN questions q ON q.id = (
  SELECT id FROM questions
  WHERE grade = 3 AND semester = 2 AND unit = SPLIT_PART(p.level_id, '-', 3)::int
  LIMIT 1
)
WHERE p.student_id IN (
  SELECT id FROM students WHERE class_id = 'uuid-class'
)
GROUP BY q.unit, q.unit_name;
```

## 三、数据导入

### 3.1 从 JSON 文件导入题目

```bash
# 1. 合并所有 JSON 文件
jq -s 'add' data/questions/*.json > all_questions.json

# 2. 用 Python 脚本导入
python scripts/import_questions.py all_questions.json
```

### 3.2 Python 导入脚本

```python
import json
from supabase import create_client

url = "https://jajxhtlwxphlpevmxewk.supabase.co"
key = "your-anon-key"
supabase = create_client(url, key)

with open("all_questions.json", "r", encoding="utf-8") as f:
    questions = json.load(f)

# 分批导入，每批 500 条
batch_size = 500
for i in range(0, len(questions), batch_size):
    batch = questions[i:i+batch_size]
    supabase.table("questions").upsert(batch).execute()
    print(f"Imported {i+len(batch)}/{len(questions)}")
```

## 四、数据导出

### 4.1 导出所有题目为 JSON

```python
import json
from supabase import create_client

supabase = create_client(url, key)

response = supabase.table("questions").select("*").execute()
questions = response.data

with open("export_questions.json", "w", encoding="utf-8") as f:
    json.dump(questions, f, ensure_ascii=False, indent=2)
```

### 4.2 导出某班级进度为 CSV

```python
import pandas as pd

response = supabase.table("progress").select("*").execute()
df = pd.DataFrame(response.data)
df.to_csv("export_progress.csv", index=False)
```
