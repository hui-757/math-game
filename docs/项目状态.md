## 五、子任务外包-ready

以下任务可以独立交给其他人/AI 完成，**只需按契约交付**：

### 任务 A：题库生成（最高优先级）
- **输入：** `接口契约.md` 第1.1/1.2节（JSON Schema + 文件命名）
- **输出：** 12个 JSON 文件 `grade01-semester1.json` ~ `grade06-semester2.json`
- **验收：** 每文件能通过 JSON 校验，每道题答案正确

### 任务 B：主题美术素材
- **输入：** 设计稿（长城红色系 / 西游记金色系）
- **输出：** 背景图（AI生成）、关卡图标（SVG）、角色形象
- **验收：** 图片尺寸一致、风格统一、适合儿童

### 任务 C：教师端统计页完善
- **输入：** `接口契约.md` 第2.2节（ClassStats 格式）
- **输出：** 完善 `TeacherView.vue`（仪表盘、柱状图、表格）
- **验收：** 传入 mock 数据能正确渲染

### 任务 D：音效素材
- **输入：** 需求描述（答对/错误/通关/背景）
- **输出：** 4~5个音效文件（MP3/WAV）或 Web Audio 代码
- **验收：** 文件大小<100KB、播放流畅
